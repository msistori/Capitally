### Quick Installation

1. Clone the repository

```bash
git clone https://github.com/msistori/capitally.git
```

2. Start the backend (Spring Boot)

```bash
cd backend
mvn spring-boot:run
```

3. Start the frontend (Angular)

```bash
cd frontend
npm install
npm start
```
